# How to Upload the Firmware

1. Install **Arduino IDE**
2. Install the required libraries
3. Select the board **ESP32 Dev Module**
4. Open `main.ino`
5. Configure WiFi and MQTT credentials
6. Upload the firmware to the ESP32

---

# Educational Purpose

This firmware is part of a pedagogical platform developed for teaching:

- Automatic Control
- Embedded Systems
- IoT Communication

Students can analyze real-time motor responses and experiment with different control strategies.
